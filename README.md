# Todo App Using TypeScript

This is a simple Todo application built with **HTML**, **Bootstrap**, **JavaScript**, and **TypeScript**.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Responsive UI with Bootstrap

## Technologies Used

- HTML5
- Bootstrap 5
- JavaScript (ES6)
- TypeScript
